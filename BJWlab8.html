The website holds a database of games. A user can create an account and link to other accounts as friends. Once a user has an account, they can create and manage personal collections of games, rate games, write reviews for games, and view other users' collections based on viewability settings. Games can be searched for to find basic descriptions and read reviews posted by users.

A complete database backend would be needed with entries for required games before deployment. Proper user authentication would need to be implemented, as well as a way for users to create their own collections.
